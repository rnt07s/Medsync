# Changelog

All notable changes to MedSync will be documented in this file.

## [1.0.0] - 2025-12-04

### Added
- Initial release of MedSync Hospital Management System
- User authentication (login/register) with JWT
- Hospital registration and management
- Doctor management for hospitals
- OPD appointment booking system
- Hospital search with interactive map
- Lab test booking functionality
- User profile management
- Dark/Light mode support
- Responsive design for all devices
- Email notifications for appointments

### Features
- **Patient Portal**: Easy appointment booking and history tracking
- **Hospital Dashboard**: Complete hospital management interface
- **Doctor Management**: Add and manage doctor profiles
- **Interactive Maps**: Find nearby hospitals using Leaflet maps
- **Real-time Updates**: Live appointment status tracking

### Tech Stack
- Frontend: React.js, Tailwind CSS, Recoil
- Backend: Node.js, Express.js, MongoDB
- Authentication: JWT with bcrypt encryption
- Maps: Leaflet.js
